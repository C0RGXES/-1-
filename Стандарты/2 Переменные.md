
# Переменные

## Имена переменных в коде

Имя переменной стараемся образовывать, исходя из терминов предметной области, чтобы было понятно её назначение

```bsl
// Неправильно
Массив = СписокСотрудников();
// Правильно
Сотрудники = СписокСотрудников();
```

Имя переменной не должно содержать в себе информации о её **типе** - это избыточная информация, которая нужна только при отладке (а в отладке мы и так видим тип значения).

```bsl
// Неправильно
МассивСотрудников = Новый Массив;
МассивСотрудников.Добавить("Иванов");
// Правильно
Сотрудники = Новый Массив;
Сотрудники.Добавить("Иванов");
// ...
// Из-за информации о типе имена переменных могут очень сильно разрастаться
МассивСтруктурСостоянийСотрудников = СостоянияСотрудников(Сотрудники);
```

Стиль написания составных слов - **только** *CamelCase* (`КаждоеСледующееСловоСБольшойБуквы`)

```bsl
// Неправильно

// Не CamelCase
Перем количество_месяцев_в_году;

// Абсолютно не читаемо
Перем количествомесяцеввгоду;

// Тоже нельзя, переменные должны именоваться с большой буквы
Перем количествоМесяцевВГоду;

// Не допускается первым символ "_"
Перем _КоличествоМесяцевВГоду;

// Правильно
Перем КоличествоМесяцевВГоду;
```

Если нужно написать имя не на русском языке, то не используем транслитерацию (транслит)

```bsl
// Несерьёзно
Sotrudnik = "Иванов";
// Хорошее имя, если нужно писать латиницей
// Employee = "Иванов";
```

Не создаём однобуквенных переменных

```bsl
// Неправильно
Перем А;

// Правильно
Перем ТекущийСотрудник;
// ...

// Допускается
// Счетчик в цикле можно
Для А = 1 По Количество Цикл
    // ...
КонецЦикла;
```

Переменные типа `Булево` именуем так, как бы именовалось истинное её значение

```bsl
// Неправильно
Перем НеДействующийСотрудник;
// Правильно
Перем ДействующийСотрудник;
```

## Форматирование

Каждую отдельную переменную размещаем на отдельной строке

```bsl
// Неправильно
Перем1 = 1; Перем2 = '20220911'; Перем3 = Истина;
// Правильно
Перем1 = 1;
Перем2 = '20220911';
Перем3 = Истина;
```

## Контекст

Правильно используем переменные ([стандарт 1С](https://its.1c.ru/db/v8std/content/639/hdoc)). Не используем глобальные переменные в модулях форм, вместо них пользуемся реквизитами формы.
